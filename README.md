# ASCII Tetris

단일 HTML 파일로 구성된 심플한 흑백 ASCII 테트리스 게임. 별도의 설치나 설정 없이 브라우저에서 즉시 실행 가능.

## 주요 특징
- **No Dependencies**: 외부 라이브러리, 이미지, CSS 파일 없이 `main.html` 파일 하나로 작동.
- **Classic ASCII Style**: 대괄호(`[]`)와 점(`..`)을 이용한 레트로 텍스트 감성.
- **Core Mechanics**: 홀드(Hold), 고스트 블록(Ghost Block), 레벨업 및 속도 증가 등 테트리스 핵심 시스템 구현.

## 실행 방법
1. `main.html` 파일을 다운로드.
2. 다운로드한 파일을 더블 클릭하여 웹 브라우저(Chrome, Edge, Safari 등)로 실행.
3. `Space` 키를 눌러 게임 시작.

## 조작 방법
- **Move**: `←`, `→` (좌우 이동)
- **Rotate**: `↑` (회전)
- **Soft Drop**: `↓` (빠르게 내리기)
- **Hard Drop**: `Space` (즉시 내리기)
- **Hold**: `C` (블록 보관)
- **Pause**: `Q` (일시 정지)
- **Resume / Restart**: `R` (재개 또는 재시작)
- **주의**: C, Q, R 키 입력 시 한글로 입력하면 안됨, 나중에(언젠가 먼 미래에) 고칠예정

## 게임 규칙
- **Level Up**: 25줄을 지울 때마다 레벨 상승 및 낙하 속도 증가.
- **Scoring**: 한 번에 많은 줄을 지울수록, 레벨이 높을수록 높은 점수 획득.
- **Game Over**: 블록이 화면 최상단에 도달하여 더 이상 생성될 수 없을 경우 종료.
